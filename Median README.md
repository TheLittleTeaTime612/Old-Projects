## About This Project:
### Description
This Project allows the user to see a very basic design of how a median is calculated. It allows for user input for the elements in predetermined size that are going to be push into the list. The program will do the calculations and then print out the results and end the program. This project was made by some vauge guidelines a couple of years ago for a assignment project. The tools involved for this project was mapping around using Visual Studios and the concepts of some of the basics with memory and pointers. 

(As of now (2026), it has been 3 years since, the list of bugs encountered is no longer remembered, however a list will be noted either as observations in the future in this README.)
