## Median Calculator
Quick Link to the cpp: [Median](https://github.com/TheLittleTeaTime612/Old-Projects/blob/main/Median.cpp)
## About This Project:
### Description
This Project allows the user to see a very basic design of how a mean is calculated. It allows for user input for the size and elements that are going to be push into the list. The program will do the calculations and then print out the results. 
### Technical Skills Involded.
- Use of arrays, function calls, memory and pointers.
- C++
